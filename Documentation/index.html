Getting started

Unzip the archive and find out the folder www. There are all the template files in this folder. You can open the following files for viewing in browser.

  1. /main — folder with main template.
  2. /admin — folder with admin template.

Main folder:

  1. /css — folder with css files.
  2. /icon — folder with favicons.
  3. /img — folder with image files.
  4. /js — folder with javascript files.

  5. index.html, index2.html, index3.html — home page options.
  6. catalog.html — catalog page.
  7. category.html, category2.html — catalog page options.
  8. details.html, details2.html, details3.html — details page options.
  9. live.html — live page.
  10. interview.html — interview page.
  11. about.html — about page.
  12. pricing.html — pricing page.
  13. profile.html — profile page.
  14. contacts.html — contacts page.
  15. privacy.html — privacy page.
  16. signin.html — signin page.
  17. signup.html — signup page.
  18. forgot.html — forgot password page.
  19. 404.html — 404 page.

Admin folder:

  1. /css — folder with css files.
  2. /icon — folder with favicons.
  3. /img — folder with image files.
  4. /js — folder with javascript files.

  5. index.html — dashboard page.
  6. catalog.html — catalog page.
  7. reviews.html — reviews page.
  8. users.html — users page.
  9. comments.html — comments page.
  10. add-item.html — item page.
  11. edit-user.html — user page.
  12. signin.html — signin page.
  13. signup.html — signup page.
  14. forgot.html — forgot password page.
  15. 404.html — 404 page.

Upload the template files (inside the folder www) to server with the help of one of the FTP-clients.

HTML structure

In my template I use Bootstrap grid, if you don't know how to use it, here is a detailed information on its use.

The standard section has the following HTML structure:

<section class="section">
  <div class="container">
    <div class="row">
      <div class="col-12">
        <h2 class="section__title">FlixTV HTML Template</h2>
      </div>
      ...
    </div>
  </div>
</section>

Header

Header have a three different styles:

header--static – header with position: relative; on the desktop version.

<header class="header header--static">
  <div class="container">
    <div class="row">
      <div class="col-12">
        <div class="header__content">
          ...
        </div>
      </div>
    </div>
  </div>
</header>

header--fixed – header with position: fixed; on the desktop version.

<header class="header header--fixed">
  <div class="container">
    <div class="row">
      <div class="col-12">
        <div class="header__content">
          ...
        </div>
      </div>
    </div>
  </div>
</header>

header--hidden – header with position: fixed; but hides when scrolling on the desktop version.

<header class="header header--hidden">
  <div class="container">
    <div class="row">
      <div class="col-12">
        <div class="header__content">
          ...
        </div>
      </div>
    </div>
  </div>
</header>

Home

  </div>

    </div>
  </div>

  <div class="home__carousel owl-carousel">

  </div>
</div>
